## ✍️ 代码要点回顾

### LeetCode 20：有效的括号
- 使用栈判断左括号入栈，右括号出栈并匹配
- 注意：判断空栈必须放在栈顶比较前面

### LeetCode 1：两数之和
- 一遍哈希：记录 num → index
- 目标值 - 当前数 = 补数，在哈希表中查找

### LeetCode 232：用栈实现队列
- 用 `in_stack` 入队，`out_stack` 出队
- 只有在 `out_stack` 为空时才将 `in_stack` 的内容倒进去（懒加载）

### LeetCode 49：字母异位词分组
- 关键：`sorted(word)` 后作为 `key`，原始字符串放入对应的 `value` 列表中归类

### LeetCode 239：滑动窗口最大值
- 维护一个单调递减的双端队列（deque）
- 每次滑动窗口时，队尾弹出小于当前值的下标，队首保存窗口最大值的下标
- 队首元素一旦滑出窗口，立刻移除
